## Objective

Upload the contents of `dist/` to the static bucket. The file `index.html` must be at the bucket root, not under a nested `dist/` folder.

Console path:

```text
S3 -> Buckets -> static frontend bucket -> Objects -> Upload
```

CLI option:

```bash
aws s3 sync dist/ s3://<S3_STATIC_BUCKET>/ --delete --region <AWS_REGION>
```
